# Lesson 1: HTML Basics and Environment Setup

## Key Skills Acquired

* **Node.js Setup:** Installed Node.js and npm on my computer. Now my system is ready to download and manage frontend libraries.
* **Basic Text Tags:** Learned how to build a website skeleton using main blocks `<div>`, headers `<h1>`-`<h3>`, and text paragraphs `<p>`.
* **Images and Buttons:** Learned how to display images with `<img>`, create web links with `<a>`, and add clickable buttons `<button>`. Also figured out how file paths work so images load correctly.

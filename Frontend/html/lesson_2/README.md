# Lesson 2: Advanced HTML Structure and Form Elements

## Key Skills Acquired

* **Lists and Menus:** Mastered creating bulleted lists (`<ul>`, `<li>`) for computer specs and dropdown selection menus (`<select>`, `<option>`) for choosing club tariffs.
* **Text Styling:** Learned how to highlight important text using `<strong>` for bold emphasis and `<span>` for future custom styling.
* **Page Layout:** Understood how to divide a website into logical zones like `<header>`, `<main>`, and `<footer>` for better code organization.
* **Advanced Inputs:** Implemented complex form fields including large text areas (`<textarea>`), time pickers (`<input type="time">`).
